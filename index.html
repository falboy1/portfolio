About

杉本 隼斗

静岡県浜松市出身です。AIやXRなど最先端技術に興味を持っていて、大学院では深層学習技術を用いた画像認識の研究を行っています。B'zが大好きで実際にライブに行ったりもしています。好きな食べ物は温泉卵で高校生の頃からほぼ毎日食べています。

Skills

Python

主に研究で使用している言語です。深層学習ではKerasとPytorchを使用しています。その他では、REST APIを使ってTwitterから対話データを収集したり、Beautiful Soup・Pandasを使用してWebページのスクレイピングとデータ操作を行いました。Flaskによるサーバサイド言語としての利用経験やeel・KivyでのGUIアプリケーションの開発経験もあります。

JavaScript

講義等での学習経験はありませんが、研究や趣味などで使用している言語です。主な使用経験はjQueryやネイティブでのDOM操作、Google App Scriptでの研究室業務の自動化です。現在はモダンな技術を学びたいと思い、ES6を意識したコーディング、React・TypeScript・Node.jsを利用したサービスの開発に挑戦中です。
